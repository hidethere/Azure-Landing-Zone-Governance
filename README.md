# Azure Landing Zone

Enterprise-grade Azure Landing Zone with Hub-and-Spoke architecture, governance, automation, and observability.

---

## Features
- Hub-and-Spoke network topology
- Azure Policy governance enforcement
- RBAC security model
- Centralized logging (Log Analytics)
- Terraform + Bicep IaC
- PowerShell + Bash automation
- Go-based compliance CLI tool
- CI/CD pipelines (GitHub Actions)
